## Spacing Scale

Declared values (multiples of 4 — 8-point system):

| Token | Value | Usage |
|-------|-------|-------|
| xs | 4px | Keyword badge gaps, icon-to-text gaps |
| sm | 8px | Intra-card padding, badge padding, button group gaps |
| md | 16px | Card padding, form field spacing, section gaps |
| lg | 24px | Section padding, spread layout gaps |
| xl | 32px | Page-level vertical rhythm, spread grid outer margin |
| 2xl | 48px | Major section breaks (e.g. spread selector to card grid) |
| 3xl | 64px | Page-level top padding |

Exceptions:
- Spread layout card touch area: minimum 44px height (accessibility touch target)
- Celtic Cross position slot: 48px × 64px minimum (10 positions must fit on one screen)
- Spread selector buttons: 40px height (compact row)

Source: default 8-point scale; exceptions from TAROT-03 success criteria (4 spread layouts, 10-position Celtic Cross)

## Color

Dark mode is the default and primary rendering mode for this application (see `globals.css` `.dark` block).

| Role | Value | Usage |
|------|-------|-------|
| Dominant (60%) | `hsl(250 45% 8%)` = `#0d0b1e` cosmic deep indigo | Page background, `bg-background` |
| Secondary (30%) | `hsl(252 35% 12%)` = `#181530` deep card surface | Card surfaces, spread slot backgrounds, `bg-surface-container` |
| Accent (10%) | `#d4a853` mystical gold | Reserved — see list below |
| Destructive | `hsl(7 100% 84%)` = `#ffb4ab` | Destructive confirmations only (no destructive actions in this phase) |

Accent (`#d4a853` gold) reserved for:
1. Active spread selector button (selected state border + text)
2. Card draw CTA button gradient highlight (`from-primary-container to-secondary-container` with gold glow shadow)
3. Keyword chip hover state border
4. Celtic Cross position number indicator
5. Metadata field labels (אסטרולוגיה, נומרולוגיה, קבלה, ארכיטיפ) icon color

Primary purple (`#ddb8ff` / `hsl(274 100% 86%)`) reserved for:
- Card name text on nebula-glow card background
- Section title text (`text-primary`)
- Focus ring on interactive elements

Source: `globals.css` dark mode variables + `tailwind.config.ts` MD3 tokens

## Spread Layout Specifications

### Spread 1: Single Card (קלף בודד)
- Layout: centered single card, full width on mobile, max-w-xs on desktop
- Shows: card name, arcana, keywords, upright meaning
- Rich fields visible inline (no modal needed for single card)

### Spread 2: Three Cards (3 קלפים)
- Layout: 3-column grid (`grid-cols-1 sm:grid-cols-3`) — existing pattern in `tarot/page.tsx`
- Position labels: עבר (Past) | הווה (Present) | עתיד (Future)
- Card click: opens `TarotCardDetailModal`

### Spread 3: Relationships (יחסים)
- Layout: 2 columns (אתה | הם) + 1 centered row (הקשר)
- 3 cards total, labeled positions
- Card click: opens `TarotCardDetailModal`

### Spread 4: Celtic Cross (צלב קלטי)
- Layout: 10 positions using CSS grid (fixed 3×5 grid with sparse placement)
- Position numbers 1–10 shown in accent gold with Hebrew position names
- ScrollArea wrapper for mobile
- Each card compact: 48px × 64px minimum touch target

## Interaction States

| State | Visual Treatment |
|-------|-----------------|
| Default (no draw) | EmptyState with `GiCrystalBall` icon, pre-draw copy |
| Loading (drawing) | `MysticSkeleton` shimmer on card slots + disabled draw button |
| Cards revealed | `ProgressiveReveal` stagger, 0.15s delay between cards |
| Card hover | `mystic-hover` class (translateY -3px + purple glow) |
| Card selected (modal) | Dialog opens with `GlassCard` variant `gold`, full metadata |
| Spread tab active | Border: `border-accent/60`, text: `text-accent`, bg: `bg-surface-container-high` |
| Spread tab inactive | `border-outline-variant/20 text-on-surface-variant` |
| Draw button pending | `opacity-60 cursor-not-allowed` + spinner icon (replace text with "שולף קלפים...") |
| Reversed card | Red rotated badge "הפוך" in top-end corner, rotated 180deg visual indicator |

## RTL Constraints

All layout uses `start`/`end` — never `left`/`right`.
- `dir="rtl"` on `<Input>` question field (already in existing page)
- Spread position labels flow right-to-left in 3-card layout
- Celtic Cross positions use `ms-` / `me-` margin utilities
- Badge flex: `flex-row-reverse` is NOT used — badges stack naturally in RTL
- Text gradient headings: `text-gradient-gold` for page title

Source: CLAUDE.md RTL & Hebrew rules
