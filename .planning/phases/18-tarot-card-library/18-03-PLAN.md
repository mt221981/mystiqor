- Inner `motion.div`: `style={{ transformStyle: 'preserve-3d', position: 'relative' }}`

   - Desktop: CSS grid `grid-template-columns: repeat(5, 1fr)` with sparse placement

     - Pos 2 (האתגר): col 2-3, row 2-3 (overlapping pos 1 — show as overlay badge, NOT rotated 90deg for MVP per Research Pitfall 6)

     - Pos 4 (העבר הקרוב): col 2-3, row 3-4
     - Pos 5 (האפשרות הטובה): col 1, row 2-3
     - Pos 6 (העתיד הקרוב): col 3-4, row 2-3
     - Pos 7 (איך אתה רואה): col 5, row 4
     - Pos 8 (איך אחרים רואים): col 5, row 3
     - Pos 9 (תקוות ופחדים): col 5, row 2
     - Pos 10 (התוצאה הסופית): col 5, row 1
   - Each position slot: minimum 48x64px touch target. Shows gold position number + Hebrew position name via Tooltip.
   - Mobile (< sm): Flatten to a numbered list instead of grid (Claude's discretion per CONTEXT.md). Each card shows its position number (gold badge) and position name.
   - Wrap in `ScrollArea` for mobile overflow.
