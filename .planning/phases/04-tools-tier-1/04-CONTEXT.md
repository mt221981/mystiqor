<decisions>
## Implementation Decisions

### Natal Chart SVG (ASTR-01)
- **D-01:** Full detail level matching BASE44 — zodiac ring (12 signs with glyphs/colors), planet positions with glyphs, aspect lines (colored by type: trine=blue, square=red, sextile=green, etc.), house cusps with numbers.
- **D-02:** SVG decomposed into 4 sub-components: `ZodiacRing.tsx`, `PlanetPositions.tsx`, `AspectLines.tsx`, `HouseOverlay.tsx` — composed by `BirthChart/index.tsx`.
- **D-03:** Port from BASE44's `BirthChart.jsx` (922 lines) — keep the same SVG geometry and math, just split and type.

### Below the Chart (ASTR-02)
- **D-04:** Four info panels below the natal chart:
  1. **AI Interpretation** — LLM-generated reading referencing specific planetary placements (ASTR-02 core deliverable)
  2. **Planet table** — list of all planets with sign, degrees, house placement
  3. **Aspect list** — table of aspects with type (trine/square/etc.) and degree orb
  4. **Quick summary** — Sun/Moon/Rising signs with 1-line descriptions

### Daily Insights (TRCK-05)
- **D-05:** Content generated by **LLM per user** — personalized based on user's zodiac sign + numerology day number + random tarot card. Costs tokens per user per day.
- **D-06:** Content sections are **user-selectable** — user chooses which modules to include in their daily insight (astro forecast, numerology energy, tarot card, personal recommendation). Default: all enabled.
- **D-07:** Generated on first visit each day, cached in `daily_insights` table with date key. Same user visiting again same day gets cached version.
- **D-08:** Daily insights page shows today's insight prominently + scrollable history of past insights.

### Personality Analysis (TOOL-06)
- **D-09:** Input via **20-question Big Five questionnaire** — 4 questions per dimension (Openness, Conscientiousness, Extraversion, Agreeableness, Neuroticism). Likert scale 1-5 (מאוד לא מסכים → מאוד מסכים).
- **D-10:** Results displayed as **Recharts RadarChart** with 5 axes + AI interpretation per dimension.
- **D-11:** Questions in Hebrew, stored in constants file. Reverse-scored items marked.

### Numerology Enhancements (NUMR-03)
- **D-12:** Numerology compatibility: accepts 2 names + 2 birth dates, calculates life path compatibility score + interpretation.
- **D-13:** Sub-number breakdown: show calculation steps (e.g., 29 → 2+9=11 → master number).

### Already Built (Verify Only)
- **D-14:** Tarot (TOOL-01) — already built at `/tools/tarot`. Verify SubscriptionGuard wrapping.
- **D-15:** Dream analysis — already built at `/tools/dream`. Verify SubscriptionGuard wrapping.

### Claude's Discretion

- Exact LLM prompt engineering for natal chart and daily insights
- Personality questionnaire phrasing (Hebrew)
- Astro calendar layout and event types
- Daily insights card design and animations
