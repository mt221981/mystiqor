    1. Create `daily-insights.ts` Zod validation:
       - Export `DailyInsightModulesSchema = z.object({ astrology: z.boolean().default(true), numerology: z.boolean().default(true), tarot: z.boolean().default(true), recommendation: z.boolean().default(true) })`
       - This controls which sections the user wants (per D-06)

    2. Create `daily-insights/route.ts` with GET handler (per D-07):
       a. Auth check
       b. Parse optional query param `modules` (JSON-encoded DailyInsightModulesSchema). If missing, default all true.
       c. Compute today's date: `new Date().toISOString().split('T')[0]` — YYYY-MM-DD UTC (per Pitfall 3, use server-side UTC consistently)
       d. **Cache check:** `SELECT * FROM daily_insights WHERE user_id=? AND insight_date=today AND mood_type='daily'` using `.maybeSingle()`
       e. **If cached:** return the existing row immediately — no LLM call (per D-07)
       f. **If not cached — gather inputs:**
         - Load user profile (birth_date) -> derive zodiac sign from month/day
         - Calculate numerology day number: `reduceToSingleDigit(day + month + currentYear)` from `@/services/numerology/calculations`
         - Select random tarot card: query `tarot_cards` table. If table has data: `.select('name, description').limit(1)` with random offset. If no tarot_cards table or empty: use a hardcoded fallback card name.
       g. **Generate via single invokeLLM call** (per Pitfall 7 — one call, not three):
         - System prompt: "אתה מייעץ מיסטי מומחה. צור תובנה יומית מקיפה בעברית עבור אדם ממזל {zodiacSign}. מספר היום הנומרולוגי שלו: {dayNumber}. קלף הטארוט ליום: {tarotCardName}."
         - Instructions in prompt: "כלול את הסעיפים הבאים (רק אם מופעלים): {if astrology: 'תחזית אסטרולוגית'} {if numerology: 'אנרגיה נומרולוגית'} {if tarot: 'הודעת הטארוט'} {if recommendation: 'המלצה אישית'}. כתוב כותרת מעוררת השראה. סיים בטיפ פעולה קונקרטי."
         - maxTokens: 1200
       h. **Cache the result:** INSERT into daily_insights: `{ user_id, insight_date: today, mood_type: 'daily', title: 'תובנה יומית', content: llmText, tarot: { card: tarotCardName }, data_sources: modules, actionable_tip: extractTipFromContent(llmText) }`
         - `extractTipFromContent`: simple regex or last paragraph extraction from LLM text
       i. **Return:** `{ data: newInsight }`
       - Error handling: try/catch, 500 on LLM failure. If cache check fails, still try to generate.

    3. Also add a second GET path or query param `?history=true` to return past insights:
       - `SELECT * FROM daily_insights WHERE user_id=? AND mood_type='daily' ORDER BY insight_date DESC LIMIT 30`
       - Return `{ data: insights }` array
       - Per D-08: page shows today prominently + scrollable history
  </action>
