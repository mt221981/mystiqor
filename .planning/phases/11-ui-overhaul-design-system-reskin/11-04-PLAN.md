**Universal replacement rules (apply to ALL files):**

Background colors:
- `bg-card` → `bg-surface-container`

- `bg-white/5` → `bg-surface-container-high`
- `bg-white/10` → `bg-surface-container-highest`

Text colors:
- `text-white` → `text-on-surface`
- `text-gray-100` → `text-on-surface`
- `text-gray-300` / `text-gray-400` → `text-on-surface-variant`
- `text-gray-500` / `text-gray-600` → `text-on-surface-variant/60`
- `text-foreground` → `text-on-surface`
- `text-muted-foreground` → `text-on-surface-variant`
- `text-purple-100` / `text-purple-200` → `text-primary-fixed`
- `text-purple-300` / `text-purple-400` → `text-primary`
- `text-purple-500` / `text-purple-600` → `text-primary-container`
- `text-indigo-400` → `text-secondary`
- `text-green-400` / `text-emerald-400` → `text-tertiary`
- `text-red-400` → `text-error`

Background accents:
- `bg-purple-600/20` / `bg-purple-600/30` → `bg-primary-container/20`
- `bg-purple-500/10` → `bg-primary/10`
- `bg-green-500/10` / `bg-emerald-500/10` → `bg-tertiary/10`

Borders:
- `border-white/5` / `border-white/10` → `border-outline-variant/10`
- `border-gray-700` / `border-gray-800` → `border-outline-variant`
- `border-purple-500` → `border-primary`

Gradients:
- `from-purple-500 to-indigo-500` → `from-primary-container to-secondary-container`
- `from-purple-400 to-indigo-400` → `from-primary to-secondary`

Shadows:
- `shadow-purple-500/20` → `shadow-primary-container/20`

Typography:
- Page titles / h1: add `font-headline`
- Body text: add `font-body` (if not already inherited)
- Labels / badges / small text: add `font-label`

Hover states:
- `hover:bg-white/5` → `hover:bg-surface-container-high`
- `hover:bg-white/10` → `hover:bg-surface-container-highest`
- `hover:text-white` → `hover:text-on-surface`
- `hover:text-purple-200` → `hover:text-primary`
