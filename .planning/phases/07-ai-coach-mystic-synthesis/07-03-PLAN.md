    ```typescript
    })
    ```

    **Define SYNTHESIS_RESPONSE_SCHEMA** (JSON Schema object for `responseSchema` param) — mirror the Zod schema structure as a plain JSON Schema object. Follow the exact pattern from career route's `CAREER_RESPONSE_SCHEMA`.

    **POST handler:**
    1. Auth check: `const supabase = await createClient()`, `const { data: { user } } = await supabase.auth.getUser()`, return 401 if no user
    2. Validate input: `SynthesisInputSchema.safeParse(await request.json())`, return 400 if invalid
    3. Fetch user data for synthesis prompt (parallel with `Promise.all`):
       - Profile: `supabase.from('profiles').select('full_name, birth_date, birth_time, gender').eq('id', user.id).maybeSingle()`
       - Recent analyses (limit 20 for on_demand, limit based on last 7 days for weekly):
         - on_demand: `supabase.from('analyses').select('tool_type, results, summary, created_at').eq('user_id', user.id).order('created_at', { ascending: false }).limit(20)`
         - weekly: same query but add `.gte('created_at', sevenDaysAgo.toISOString())`
         - `sevenDaysAgo = new Date(Date.now() - 7 * 24 * 60 * 60 * 1000)`
       - Goals: `supabase.from('goals').select('title, status, category, progress').eq('user_id', user.id).eq('status', 'active').limit(10)`
       - Moods: `supabase.from('mood_entries').select('mood_score, energy_level, created_at').eq('user_id', user.id).order('created_at', { ascending: false }).limit(14)`
    4. Build `inputSources` array: list of tool_type strings from fetched analyses (unique), e.g. `['astrology', 'numerology', 'drawing']`
    5. Check minimum data: if analyses count < 2, return 400 with `{ error: 'נדרשים לפחות 2 ניתוחים כדי ליצור סינתזה' }`
    6. Build Hebrew prompt:
       - For on_demand:
         ```
         צור סינתזה מיסטית הוליסטית עבור המשתמש הבא. שלב את כל הנתונים מהניתוחים השונים לתמונה אחת מאוחדת.

         פרטי המשתמש:
         שם: ${profile?.full_name ?? 'לא ידוע'}
         תאריך לידה: ${profile?.birth_date ?? 'לא ידוע'}

         ניתוחים שבוצעו (${analyses.length} ניתוחים מסוגים: ${inputSources.join(', ')}):
         ${analyses.map(a => `- ${a.tool_type}: ${a.summary ?? JSON.stringify(a.results).slice(0, 300)}`).join('\n')}

         יעדים פעילים: ${goals?.map(g => g.title).join(', ') ?? 'אין'}

         מצב רוח ממוצע (14 ימים): ${avgMood}/5

         צור פרופיל אישיות מאוחד שמשלב נתונים מכל הכלים, תחזיות עתידיות והמלצות מעשיות.
         חשוב: ציין במפורש את הקשרים בין הכלים השונים (למשל "מספר מסלול החיים 7 שלך מתיישב עם שבתאי במזל דלי...").
         ```
       - For weekly, add additional instruction:
         ```
         זוהי סינתזה שבועית. כלול גם:
         - ניתוח שימוש: באילו כלים השתמשת הכי הרבה השבוע
         - אינטגרציה מעשית: הצעות ליישום התובנות בחיי היום-יום עם רמות קושי
         - סיכום תקופה: סיכום קצר של השבוע שעבר
         ```
    7. Call `invokeLLM<SynthesisResponse>`:
       ```typescript
       const response = await invokeLLM<SynthesisResponse>({
         prompt: synthesisPrompt,
         systemPrompt: 'אתה מומחה לסינתזה מיסטית הוליסטית. שלב ידע מאסטרולוגיה, נומרולוגיה, ניתוח ציורים, גרפולוגיה וכלים נוספים לתמונה אחת שלמה. ענה בעברית בלבד.',
         responseSchema: SYNTHESIS_RESPONSE_SCHEMA,
         zodSchema: SynthesisResponseSchema,
         maxTokens: 6000,
         userId: user.id,
       })
       ```
    8. Validate: check `response.validationResult?.success`, if false return 500 with error
    9. Extract validated data: `const result = response.validationResult!.data as SynthesisResponse`
    10. Save to analyses table:
        ```typescript
        const row: TablesInsert<'analyses'> = {
          user_id: user.id,
          tool_type: 'synthesis',
          input_data: { sources: inputSources, type: synthesisType },
          results: {
            personality_profile: result.personality_profile,
            predictive_insights: result.predictive_insights,
            recommendations: result.recommendations,
            ...(synthesisType === 'weekly' ? {
              usage_analysis: result.usage_analysis,
              practical_integration: result.practical_integration,
              period_summary: result.period_summary,
            } : {}),
          },
          summary: result.personality_profile.summary.slice(0, 500),
        }
        const { data: analysis, error } = await supabase.from('analyses').insert(row).select().single()
        ```
    11. Return `NextResponse.json({ data: { ...result, analysis_id: analysis.id } })`
    12. Wrap in try/catch, return 500 with Hebrew error on failure
