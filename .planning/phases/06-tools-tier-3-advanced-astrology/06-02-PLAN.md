<objective>
Build the Transits and Solar Return tools — two ephemeris-dependent astrology features. Transits shows current planetary positions and their aspects to the user's natal chart. Solar Return generates an annual forecast based on the exact moment the Sun returns to its natal position.

Purpose: These are the core advanced astrology features that differentiate MystiQor from basic horoscope apps. Both depend on Plan 06-01's ephemeris adapter.
Output: Two complete API routes + two full UI pages with SubscriptionGuard.
</objective>
