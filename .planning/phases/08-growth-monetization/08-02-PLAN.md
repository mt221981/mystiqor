Full implementation:
1. Import `NextRequest`, `NextResponse` from `next/server`

5. Instantiate: `const stripe = new Stripe(process.env.STRIPE_SECRET_KEY!)`

6. Export async POST handler:

**Step A — Signature verification:**
```
const body = await request.text(); // NEVER request.json() — Pitfall 1
const sig = request.headers.get('stripe-signature') ?? '';
let event: Stripe.Event;
try {
  event = stripe.webhooks.constructEvent(body, sig, process.env.STRIPE_WEBHOOK_SECRET!);
} catch {
  return NextResponse.json({ error: 'Invalid signature' }, { status: 400 });
}
```

**Step B — Idempotency check (insert BEFORE processing — Pitfall 2):**
```
const supabase = createAdminClient();
const { error: dupError } = await supabase
  .from('processed_webhook_events')
  .insert({ stripe_event_id: event.id, event_type: event.type });
if (dupError) {
  // unique constraint violation = already processed
  return NextResponse.json({ received: true });
}
```

**Step C — Event handling (switch on event.type):**

Case `checkout.session.completed`:
- Extract `session = event.data.object as Stripe.Checkout.Session`
- Get `userId = session.metadata?.user_id` and `planId = session.metadata?.plan_id`
- Guard: if (!userId || !planId) break
- Upsert to subscriptions: `{ user_id: userId, plan_type: planId, status: 'active', stripe_customer_id: session.customer as string, stripe_subscription_id: session.subscription as string, analyses_limit: planId === 'basic' ? 20 : -1, analyses_used: 0, start_date: new Date().toISOString() }` with `onConflict: 'user_id'`

Case `customer.subscription.updated`:
- Extract `sub = event.data.object as Stripe.Subscription`
- Update subscriptions: `{ status: sub.status, cancel_at_period_end: sub.cancel_at_period_end }` where `stripe_subscription_id = sub.id`

Case `customer.subscription.deleted`:
- Extract `sub = event.data.object as Stripe.Subscription`
- Update subscriptions: `{ status: 'cancelled', plan_type: 'free', analyses_limit: 3, analyses_used: 0 }` where `stripe_subscription_id = sub.id`

Case `invoice.payment_failed`:
- Extract `invoice = event.data.object as Stripe.Invoice`
- Get stripe_customer_id from `invoice.customer as string`
- Query subscriptions for user by `stripe_customer_id`, join to profiles to get email and name
- If user found, call `sendPaymentFailedEmail(email, name, amountInShekel)` where amount is `(invoice.amount_due ?? 0) / 100`
- Wrap email send in try/catch — email failure should NOT cause webhook to return non-200

**Step D — Return 200:**

return NextResponse.json({ received: true });
