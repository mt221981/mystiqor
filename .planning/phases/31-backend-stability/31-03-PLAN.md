Add Zod LLM response validation to the three tools that currently return free-text without any schema: tarot, palmistry, and dream. Create the missing `palmistry.ts` schema. Output: `palmistry.ts` schema file, updated `index.ts`, and three updated routes with `responseSchema` + `zodSchema` + `validationResult.success` checks.
