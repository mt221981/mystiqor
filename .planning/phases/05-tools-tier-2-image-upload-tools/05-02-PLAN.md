<!-- DigitalCanvas reference from BASE44 source -->
From github-source/src/components/DigitalCanvas.jsx (475 lines):
- Canvas is 800x600 fixed pixels
- Uses getCanvasCoordinates() for scale normalization: canvas.width / rect.width
- Touch events: startDrawing, draw, stopDrawing with e.preventDefault()
- Stroke metadata: { total_strokes, total_drawing_time, average_speed, average_pressure, erasures, canvas_dimensions, drawing_date }
- Export: canvas.toBlob(callback, 'image/jpeg', 0.85) → File → onSave(file, metadata)
- Tools: pen (black), eraser (white fill), undo (pop last stroke), clear (wipe all)
- CSS: touch-action: none on canvas element

TypeScript interface for DigitalCanvas:
```typescript
interface CanvasProps {
  onSave: (file: File, metadata: DrawingMetadata) => void
  onCancel: () => void
}
interface DrawingMetadata {
  total_strokes: number
  total_drawing_time: number
  average_speed: number
  average_pressure: number
  erasures: number
  canvas_dimensions: { width: number; height: number }
  drawing_date: string
}
```

From the drawing page (Plan 01 output):
- Currently only has file upload mode

CRITICAL Pitfall 3: Use 'image/jpeg' at 0.85 quality (NOT 'image/png') to keep exported canvas under 4.5MB Vercel body limit.
