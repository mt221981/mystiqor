Scoring logic (inline helper function, pure, exportable for test):
```typescript
export function calculateCombinedScore(numerologyScore: number, astrologyScore: number): number {
  const raw = numerologyScore * 0.40 + astrologyScore * 0.60
  return Math.min(100, Math.max(0, Math.round(raw)))
}
```

Route logic:
1. Auth check
2. Validate
3. Parallel calculation:
```typescript
const [numResult, chart1, chart2] = await Promise.allSettled([
  Promise.resolve(calculateNumerologyCompatibility(person1, person2)),  // sync, wrap in resolved Promise
  person1.latitude && person1.longitude
    ? calculateBirthChart({ birthDate: person1.birthDate, birthTime: person1.birthTime, latitude: person1.latitude, longitude: person1.longitude })
    : Promise.resolve(null),
  person2.latitude && person2.longitude
    ? calculateBirthChart({ ... })
    : Promise.resolve(null),
])
```
4. Compute astrology element score (if both charts available):
```typescript
const c1 = chart1.status === 'fulfilled' ? chart1.value : null
const c2 = chart2.status === 'fulfilled' ? chart2.value : null
const astrologyScore = c1 && c2
  ? Math.round(
      ELEMENT_COMPAT[getElement(c1.sunSign)]?.[getElement(c2.sunSign)] * 0.5 +
      ELEMENT_COMPAT[getElement(c1.moonSign)]?.[getElement(c2.moonSign)] * 0.3 +
      ELEMENT_COMPAT[getElement(c1.risingSign)]?.[getElement(c2.risingSign)] * 0.2
    )
  : 65  // fallback when no location provided
```
5. `const numerologyScore = numResult.status === 'fulfilled' ? numResult.value.overallScore : 60`
6. `const totalScore = calculateCombinedScore(numerologyScore, astrologyScore)`
