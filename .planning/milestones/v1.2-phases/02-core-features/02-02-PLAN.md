Input schema (define inline in this file if not in validations/analysis.ts):
```typescript
const NumerologyInputSchema = z.object({
  fullName: z.string().min(1, 'שם מלא חובה').max(100, 'שם ארוך מדי'),
  birthDate: z.string().regex(/^\d{4}-\d{2}-\d{2}$/, 'פורמט תאריך לא תקין — נדרש YYYY-MM-DD'),
})
```

Route body:
1. `const supabase = await createClient()` + auth check (return 401 if no user)
2. Parse + validate body with NumerologyInputSchema.safeParse
3. `const numbers = calculateNumerologyNumbers(parsed.data)` — no try/catch needed here, it's pure math
4. Generate AI interpretation via `invokeLLM`:
   - systemPrompt: "אתה מומחה נומרולוגיה עברית. תן פרשנות מותאמת אישית ב-3 פסקאות קצרות. התמקד בנקודות החיובית ובכוחות. אל תיתן ניבוי גורל — רק פוטנציאלים."
   - userPrompt: `"מספרי הנומרולוגיה של ${numbers.name}: נתיב חיים ${numbers.life_path}, גורל ${numbers.destiny}, נשמה ${numbers.soul}, אישיות ${numbers.personality}, שנה אישית ${numbers.personal_year}. פרש את המשמעות המשולבת."`
5. Save to analyses: `supabase.from('analyses').insert({ user_id: user.id, tool_type: 'numerology', input_data: parsed.data, results: { ...numbers, interpretation: aiText }, summary: `נתיב חיים: ${numbers.life_path}` }).select('id').single()`
6. Return `NextResponse.json({ data: { ...numbers, interpretation: aiText, analysis_id: analysis?.id } })`
7. Catch block: `return NextResponse.json({ error: 'שגיאה בחישוב נומרולוגי' }, { status: 500 })`

 */

Input schema:
```typescript
const TarotInputSchema = z.object({
  spreadCount: z.union([z.literal(1), z.literal(3), z.literal(5)]).default(3),
  question: z.string().max(300).optional(),
})
```

Logic:
1. Auth check (401 if no user)
2. Validate input
3. Fetch cards: `const { data: allCards } = await supabase.from('tarot_cards').select('*')` — handle null/empty with fallback `[]`
4. Shuffle + draw: `const shuffled = [...(allCards ?? [])].sort(() => Math.random() - 0.5); const drawn = shuffled.slice(0, parsed.data.spreadCount)`
5. If drawn.length === 0, return `NextResponse.json({ error: 'אין קלפי טארוט זמינים. יש להריץ את ה-seed.' }, { status: 500 })`
6. AI interpretation: invokeLLM with prompt including card names (Hebrew) + question if provided
7. Save to analyses + return
